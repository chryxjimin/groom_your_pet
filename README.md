# Groom Your Pet!

Groom Your Pet! is a Rails library for groomers who have to make appointments for customers.

## Installation

Add this line to your application's Gemfile:

```rails
gem 'groom_your_pet'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install groom_your_pet
